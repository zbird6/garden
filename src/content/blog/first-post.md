## 欢迎来到 花园

这是一个基于 Astro 和 Vue 3 构建的个人静态网站 Demo，集成了博客、图片展示和原生游戏功能。

### 主要特性

1. **静态博客** - 使用 Astro Content Collections 管理 Markdown 内容
2. **图片展示** - Vue 3 组件实现图片轮播和原文跳转
3. **原生游戏** - HTML5 Canvas + 原生 JavaScript 实现
4. **Netlify 部署** - 零配置部署，支持自动构建和 CDN 加速

### 技术栈

- **Astro** - 核心框架，静态渲染
- **Vue 3** - 轻交互组件
- **Tailwind CSS** - 样式框架
- **Vite** - 构建工具
- **TypeScript** - 类型支持
